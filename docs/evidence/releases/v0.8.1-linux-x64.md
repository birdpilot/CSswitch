## 分支与后续开发

发布 source 由 immutable tag 和 GitHub Release 保留。发布后核对发现原工作分支名为 `codex/v090-linux-x64`，但其实际版本、tag 和附件均为 0.8.1；该误导性的本地/远端工作分支及临时 worktree 已在 2026-07-23 清理，没有删除 tag、Release 或附件。

Linux 0.8.1 因此是冻结的 prerelease，不是继续追加提交的活动分支。后续 Linux 工作应从当时最新 `origin/main` 新建明确命名的分支，审阅并前移 Linux platform delta；不得把已落后 macOS 主线的 0.8.1 tag 直接当作新版本开发基线。

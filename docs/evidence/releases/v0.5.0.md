## 签名、notarization 与 Gatekeeper

| 检查 | 结果 | 准确含义 |
|---|---|---|
| `codesign --verify --deep --strict --verbose=4 CSSwitch.app` | 通过 | app seal / 内嵌签名结构有效 |
| `codesign -dvvv CSSwitch.app` | `Signature=adhoc`, `TeamIdentifier=not set` | 不是 Developer ID 团队签名 |
| `spctl --assess --type open --context context:primary-signature ...dmg` | exit 3，`rejected`, `source=no usable signature` | Gatekeeper 未接受最终 DMG |
| `xcrun stapler validate ...dmg` | exit 65，`does not have a ticket stapled` | 没有 stapled notarization ticket |

结论：v0.5.0 是 ad-hoc signed、未建立 Developer ID / notarization / Gatekeeper 接受的公开附件。用户首次打开可能需要 Finder 右键“打开”。不得把 `codesign --verify` 通过写成“已签名公证”。
